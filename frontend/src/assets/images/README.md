# Instruções para a Logo

Por favor, salve a imagem da logo anexada nesta pasta com o nome:

**`praxis-logo.png`**

A imagem deve ser a logo oficial da Praxis (com as listras amarelas e pretas formando escadas e o texto "PRAXIS" abaixo com o slogan "SIMULE | EVOLUA | PRATIQUE").

A logo será exibida no header de todas as páginas:
- Landing
- Home
- Profile
- Login

Tamanho recomendado: altura de aproximadamente 40-50px, com largura proporcional.
